What is ChatWork ?

ChatWork is a business chat app that makes collaborative teamwork more efficient. It brings together text, voice and video chat with task management and file sharing, all in one convenient place. It can turn a contact into a connection with one click, or transform tasks into teamwork. ChatWork works with most modern Web browsers and requires no additional software to install.

What is Bitbucket ?

Bitbucket is a web-based hosting service for projects that use either the Mercurial (since launch) or Git revision control systems. Bitbucket offers both commercial plans and free accounts. It offers free accounts with an unlimited number of private repository. Bitbucket is written in Python using the Django web framework.

Main Responsibility :

1. Creating a integration of these 2 applications. This application should notify all the team members whenever there is a chnage in the repository. The Application provides information such as branch name, date and time, commit message.

2. It also notifies specific user, if there is a conflict that has occurred.

3. Creating a shell Script which runs and installs this application within desired Git repository.

Technologies : shell Scripting, Git (version-control-system), ChatWork API

Survey on Cloud Computing in Agricultural Development

I finished this projects as a part of my final year project at KLS GIT.

Duration : February 2014 - June 2014

Performed a survey on cloud computing in agricultural development and studied about agriculture -cloud which is based on MAD cloud architecture.

Survey Result :

Agriculture has traditionally been maintained by families and communities where the passing on the sharing of knowledge is regarded as very important. The accumulation and sharing of knowledge has resulted in better overall efficiency and productivity. Agriculture is the embodiment of a large amount of ancient knowledge If the leverage effects of IT can be developed, then we should be able to bring about a further leap in agriculture.

It goes without saying that cloud computing can support this process. Instead, one might say that the mechanisum of cloud computing is highly suited to the task of handing down human knowledge to later generation.

The above model bridges the information gap within and outside the nation. In Indian agriculture the suggested model is like the pilot project. This will definately have a positive impact in the over all economic development.
